Syllabus of GCP DevOps Course

Download Curriculum

1. Introduction to GCP DevOps

  • Overview of Google Cloud Platform (GCP) and DevOps
  • Key Concepts and Components of GCP
  • GCP DevOps Services
  • Benefits of Using GCP DevOps
  • Understanding DevOps Culture and Practices
  • Introduction to Agile and Scrum
  • GCP Architecture Overview

2. Version Control with Git

2.1 Getting Started with Git

  • Introduction to Git and Version Control
  • Setting up Git Environment
  • Basic Git Commands
  • Creating and Cloning Repositories
  • Understanding Git Workflow
  • Branching and Merging
  • Working with Remote Repositories

2.2 Advanced Git Techniques

  • Understanding Git Rebase
  • Cherry Picking Commits
  • Stashing Changes
  • Resolving Merge Conflicts
  • Utilizing Git Hooks
  • Tagging and Releases

3. Continuous Integration and Continuous Deployment (CI/CD)

3.1 Introduction to CI/CD

  • Concepts of Continuous Integration (CI)
  • Concepts of Continuous Deployment (CD)
  • Benefits and Challenges of CI/CD
  • Google Cloud Build Overview
  • Setting Up Build Pipelines with Cloud Build
  • Creating Release Pipelines using Cloud Deploy
  • Pipeline Stages and Jobs

3.2 Advanced CI/CD Concepts

  • Pipeline Variables and Secrets Management
  • Multi-Stage Pipelines
  • Deployment Strategies (Canary, Blue/Green)
  • Automating Unit and Integration Tests
  • Managing Pipeline Performance
  • Integrating with External Tools and Services

4. Infrastructure as Code (IaC)

4.1 Introduction to IaC

  • Concepts of Infrastructure as Code
  • Benefits of IaC
  • Google Cloud Deployment Manager Overview
  • Creating and Managing Deployment Manager Templates
  • Introduction to Terraform
  • Writing and Managing Terraform Scripts

4.2 Advanced IaC Practices

  • Terraform Modules and Workspaces
  • Managing Terraform State Files
  • Using GCP Tools for IaC
  • Integrating IaC with CI/CD Pipelines
  • Best Practices for IaC
  • Handling Dependencies and State Management

5. Monitoring and Logging

5.1 Google Cloud Operations Suite

  • Introduction to Google Cloud Operations Suite
  • Setting Up Monitoring and Alerts
  • Creating and Managing Alerts in Cloud Monitoring
  • Analyzing Logs and Metrics with Cloud Logging
  • Visualizing Data with Google Cloud Monitoring Dashboards
  • Integrating Cloud Monitoring with Other Services

5.2 Application Performance Monitoring

  • Overview of Google Cloud Trace and Debugger
  • Instrumenting Applications for Monitoring
  • Creating and Analyzing Custom Metrics
  • Setting Up and Managing Application Alerts
  • Analyzing Application Performance and Dependencies
  • Visualizing and Diagnosing Issues

6. Security and Compliance

6.1 Security Best Practices

  • Implementing Security in CI/CD Pipelines
  • Managing Secrets and Credentials with Secret Manager
  • Access Control and IAM Management in GCP
  • Compliance and Governance in GCP
  • Handling Security Vulnerabilities
  • Auditing and Reporting in GCP

6.2 Compliance and Governance

  • Using GCP Organization Policies for Governance
  • Implementing Compliance Controls in GCP
  • Best Practices for Security and Compliance
  • Managing Regulatory Compliance Requirements
  • Automating Compliance Checks in GCP

7. Capstone Project

  • Project Overview and Requirements
  • Designing CI/CD Pipelines in GCP
  • Implementing Infrastructure as Code using Terraform
  • Monitoring and Logging Implementation in GCP
  • Ensuring Security and Compliance in GCP
  • Project Presentation and Review
